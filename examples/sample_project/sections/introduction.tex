本项目用于测试 \texttt{latex-to-word} 流水线是否能够正确处理：

\begin{itemize}
    \item 标题层级与目录；
    \item 图片与图题；
    \item 普通表格与长表格；
    \item 数学公式及其编号；
    \item 文献引用与文末参考文献；
    \item \verb|\ref|、\verb|\eqref|、\verb|\autoref|、\verb|\cref| 等交叉引用。
\end{itemize}

为了便于检查，本示例包含一个公式引用 \eqref{eq:state}、一个图片引用 \autoref{fig:sample-curve}、
以及一个表格引用 \cref{tab:params,tab:long}。
